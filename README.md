# Sitnikov
Управление доставкой
Снабженец заключает договора с поставщиками, формирует заявки, контролирует выполнение заказа, принимает товар на склад, списывает товар, выбирает поставщиков. Логист организует доставку товаров – назначает автомобиль, водителя, формирует маршрут с учётом грузоподъёмности автомобиля и параметров груза. Администратор управляет учётными записями.
